# Review

## Recurring Issues

- 问界面“显示什么”时，优先用 show 或 showing，不要机械套用 there are
- 讲对象“有属性”时，优先用 has，不要用 there are
- 讲 demo 口播时，优先拆成短句，减少中文直译长句
- 高概率把单数/复数和主谓一致说乱：there is a new field, there are new sections, warnings were identified
- 高概率把 UI 状态说成中文直译：button active, validate name, pop up toast, show warning count
- 高概率在动作句里漏连接词：click name open panel, warning objectId is empty, field changed form ...

## High-Frequency Errors From English Notes

- 拼写反复错：detail, single, assumption, warning, expression, active, display, behavior, template, attributes, automatically
- UI 动作模板不稳：click the name to open ..., open the detail panel, switch to the tab, display a warning card
- there are / has / shows 混用：说页面上“有几块区域”用 there are，说对象“有属性”用 has，说界面“显示内容”用 shows
- 单复数和时态不稳：a toast pops up, warnings and assumptions were identified, users can click, the button is enabled
- 结尾提问不稳：固定改成 Any questions? 或 Do you have any questions?
- 产品术语最好固定：detail panel, warning card, diagnostics tab, single-select menu, valid name, expression error

## Common Demo Rewrites

- 错：there are a new icon
	- 改：there is a new icon

- 错：user can click the name, to wiew detile
	- 改：Users can click the name to view the detail panel.

- 错：the apply button active
	- 改：The Apply button is enabled.

- 错：type a validate name
	- 改：Enter a valid name.

- 错：we can get the correct result from expression
	- 改：We can see the correct expression value.

- 错：do you have any question?
	- 改：Do you have any questions?

## Quick Practice

- After reconciliation, a toast pops up showing how many warnings and assumptions were identified.
- Users can only select one material.
- The Warning section shows all items with a severity of Warning.
- Users can click the name to open the detail panel.
- There is a new warning icon on the toolbar.
- The Apply button is enabled after we enter a valid name.

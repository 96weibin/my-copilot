# Patterns

## Not Mastered

- there are vs has vs shows
  - note: there are 表示某处存在某物；has 表示对象拥有属性；shows 表示界面展示信息
  - example: There are three sections in this tab. / The Tank template has two string attributes. / The toast shows how many warnings were identified.
  - status: learning

- can only vs can have only one
  - note: can only 更强调限制；can have only one 更强调数量限制
  - example: Users can only select one material. / Each material can have only one parent.
  - status: learning

- showing how many ... were identified
  - note: 适合表达“界面显示识别出了多少条数据”
  - example: A toast pops up showing how many warnings and assumptions were identified.
  - status: learning

- changed from A to B
  - note: 适合表达字段或 UI 从一种形式改成另一种形式
  - example: The field has been changed from multiple checkboxes to a single-select menu.
  - status: learning

- demo narration: user can click ... to open ...
  - note: 讲 UI 动作时优先用固定模板，不要临时直译中文顺序
  - example: Users can click the name to open the detail panel.
  - status: learning

- demo narration: there is/are + noun, not there are a new ...
  - note: 不说 there are a new icon/field/tab；单数用 there is a new ...，复数用 there are new ...
  - example: There is a new warning icon on the toolbar. / There are three sections in this tab.
  - status: learning

- demo narration: natural closing questions
  - note: 结尾优先用 Any questions? 或 Do you have any questions?，不要用 any question
  - example: That is all for my demo. Any questions?
  - status: learning

- adjective vs verb in UI status
  - note: 按钮状态常用 enabled/disabled，不说 the apply button active；名字校验常用 valid，不说 validate name
  - example: The Apply button is enabled after we enter a valid name.
  - status: learning

- result wording: see/show/display instead of get
  - note: 讲界面结果优先用 see, show, display；不要频繁用 get the correct result from expression
  - example: We can see the correct expression value. / The page shows the correct result.
  - status: learning

- roll ... over
  - note: 把未完成的工作顺延到下一个 iteration 或 sprint
  - example: If this story is not done by Friday, we can roll it over to the next sprint.
  - template: roll [story/task/work] over to the next [iteration/sprint]
  - status: learning

- walk us through ...
  - note: 请某人带大家过一遍某个内容、计划、方案或 board
  - example: Can you walk us through the plan for the next iteration?
  - template: walk us through the [plan/design/board/timeline]
  - status: learning

- priorities
  - note: 优先事项，常用于讨论下一轮或当前最重要的事
  - example: Let's align on the top priorities for the next iteration.
  - collocation: top priorities / next iteration priorities
  - status: learning
